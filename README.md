# Enterprise-Network-Infrastructure
Cisco Packet Tracer lab involving Static Routing, DHCP Relay, DNS, and HTTP services.
This is a network lab I built in Cisco Packet Tracer. I wanted to see how a real company's network works by connecting different departments to a central data center.
What I did in this lab:
* Routing: I set up static routes so all departments (HR, IT, Sales) can talk to each other and reach the server.
* Automatic IP (DHCP): I configured the router as a Relay Agent (IP Helper) so all computers get their IP addresses automatically from the main server.
* Web & DNS: I set up a web server and a DNS. You can type lab.sahib.com in any PC's browser to see my personal web page.
* Printers: Added networked printers for each office so everyone can print documents over the network.
* Naming: I gave every router and switch a proper name like R-GATEWAY-01 to keep everything organized.
Network Details:
* IT Dept: 192.168.10.0
* Sales Dept: 192.168.20.0
* HR Dept: 192.168.30.0
* Server: 192.168.100.2
